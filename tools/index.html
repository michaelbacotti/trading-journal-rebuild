Broker

The journal uses a major retail brokerage that supports SPX and XSP options with commission-free closing trades. The specific broker is not disclosed because the trade structure is broker-agnostic. The methodology applies to any broker that supports CBOE-listed index options.

Data & Charts

Market data, charts, and options chains are pulled from the broker's platform. Historical volatility comparisons and IV-rank readings are calculated using the broker's tools. Public.com is used for real-time quotes on sector ETFs and certain less-liquid tickers.

Calculators

Position sizing follows a fixed-fractional model (see the playbook). The journal uses a spreadsheet for expected-value calculations across spread structures, and the broker's built-in options calculator for single-leg Greeks.

Notes & Records

Every trade is recorded in a structured format — instrument, strike(s), expiration, premium, target, stop, entry reasoning, and exit reasoning — that mirrors the trade log on this site. The records are kept in plain markdown so they can be version-controlled and audited.

How to use the tools

Readers new to options should start with a visualization tool (OptionsStrat is the one used in this journal) and a broker that supports the underlying instruments they want to trade. The visualization tool lets you see how a specific structure behaves across different price paths before you commit capital. The broker needs to support the instrument class (equity index options, single-stock options, ETF options) and ideally offer commission-free closing trades because the playbook's exit strategy often involves closing spreads early.

For readers who want to track their own trades the way the journal does, the data and calculation tools are sufficient to start. Most brokers provide real-time options chains, IV-rank readings, and basic options calculators. A spreadsheet is sufficient for position sizing and expected-value calculation; no specialized software is required. The journal's full tool stack is documented on the relevant pages.

Limitations of the tools

Visualization tools show what a structure would look like at expiration or at a specific point in time, but they do not predict how the underlying will move. Probability of profit (POP) values from options calculators assume a lognormal distribution of returns, which is a useful approximation but not a perfect description of equity index returns (which exhibit fat tails, especially during volatility events). The Greeks shown by options calculators are correct for small moves but degrade in accuracy for large moves.

The journal uses the tools as inputs to decision-making, not as decision-making substitutes. The playbook's rules and the trader's judgment are the final inputs. Tools inform; humans decide.
